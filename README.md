# 🤖 LinguiniBot AI

## 📌 Introducción

**LinguiniBot AI** es un chatbot educativo diseñado para apoyar el aprendizaje de lenguas extranjeras, especialmente el inglés, en contextos académicos en Colombia. Su objetivo principal es servir como una herramienta interactiva que facilite la práctica del idioma en diferentes niveles, desde básico (A1–A2) hasta intermedio y avanzado (B1–C1).

El sistema permite a los usuarios interactuar en tiempo real con un modelo de inteligencia artificial, promoviendo el desarrollo de habilidades como comprensión lectora, escritura y práctica conversacional.

---

## 🎯 Objetivo del Proyecto

Brindar una solución tecnológica accesible que:

- Refuerce el aprendizaje del inglés en estudiantes.
- Permita practicar conversaciones en tiempo real.
- Apoye procesos educativos tanto autónomos como guiados.
- Sirva como complemento a metodologías tradicionales de enseñanza.

---

## 🧠 Enfoque Académico

LinguiniBot AI está orientado al contexto educativo colombiano, alineándose con niveles del Marco Común Europeo de Referencia para las Lenguas (MCER):

- **Nivel Básico (A1 - A2):**
  - Vocabulario simple
  - Frases cotidianas
  - Preguntas y respuestas básicas

- **Nivel Intermedio (B1 - B2):**
  - Conversaciones más estructuradas
  - Comprensión de textos simples
  - Expresión de ideas

- **Nivel Avanzado (C1):**
  - Conversaciones complejas
  - Argumentación
  - Uso contextual del idioma

## ⚙️ Tecnologías Utilizadas

- **Frontend:**
  - HTML5
  - CSS3
  - JavaScript

- **API de Inteligencia Artificial:**
  - Modelo generativo (Utilizando arquitectura NPL)

---

## 💡 Funcionalidades

- 💬 Chat interactivo en tiempo real
- ⌨️ Envío de mensajes mediante botón o tecla Enter
- 🧠 Respuestas generadas por IA
- 📜 Historial dinámico de conversación
- 🔄 Auto-scroll en mensajes
- ❌ Manejo básico de errores

---

## 🏗️ Arquitectura

El sistema sigue una estructura modular simple:

- **Interfaz de Usuario (UI):**
  Manejo de visualización y estilos del chatbot.

- **Lógica de Cliente (JavaScript):**
  Control de eventos, renderizado de mensajes y flujo de interacción.

- **Servicio Externo (API):**
  Procesamiento del lenguaje natural mediante IA.
